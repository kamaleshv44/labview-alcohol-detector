# LabVIEW Alcohol Level Detector with myRIO

Internship project (2022-2023) for breath alcohol detection using LabVIEW, myRIO, and MQ-3 sensor.

## Overview
- **Problem**: Drunk driving accidents.
- **Solution**: Threshold-based system; green LED (safe), red LED/engine lock (detected).
- **Components**: MQ-3 Sensor, myRIO, LEDs, Breadboard.
